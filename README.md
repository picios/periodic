# periodic
Periodic table in React.js
I used [https://github.com/Bowserinator/Periodic-Table-JSON](https://github.com/Bowserinator/Periodic-Table-JSON) as an input data set.

## Install

```
npm install
```

## Run
```
npm run watch
```
then go to
http://localhost:8080

## Build
```
npm run build
```
